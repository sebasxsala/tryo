## Installation

Install Runtry using your favorite package manager:

<CodeGroup>
```bash npm
npm install runtry
```

```bash pnpm
pnpm add runtry
```

```bash yarn
yarn add runtry
```

```bash bun
bun add runtry
```
</CodeGroup>

## Basic Usage

Runtry provides a simple way to handle async operations safely.

### Using `run` directly

For simple use cases, you can use the `run` function directly:

```typescript
import { run } from "runtry";

const result = await run(
  () => fetch("https://api.example.com/data").then((res) => res.json()),
  {
    retries: 3,
    timeout: 5000,
  }
);

if (result.ok) {
  console.log("Success:", result.data);
} else {
  console.error("Failed:", result.error);
}
```

### Using `createRunner`

For more advanced configurations and reuse, create a `Runner` instance:

```typescript
import { createRunner } from "runtry";

// Create a reusable runner with default configuration
const runner = createRunner({
  retries: 2,
  timeout: 10000,
  // Global error handling rules
  onError: (error) => {
    // Log to your monitoring service
    console.error("Global error handler:", error);
  }
});

// Use it across your application
const result = await runner.run(() => complexOperation());
```
