# Tryo Documentation

This repository contains the source code for the [Tryo](https://github.com/sebastiansala/tryo) documentation, built with [Mintlify](https://mintlify.com/).

Tryo is a TypeScript library for resilient task execution, featuring retries, circuit breakers, and concurrency control.

## Running Locally

1. Install the Mintlify CLI:
   ```bash
   npm i -g mint
   ```

2. Run the development server:
   ```bash
   mint dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Structure

The documentation is organized by language:

### English (Root)
- `api-reference`: API documentation
- `concepts`: Core concepts (Result Type, Error Handling, etc.)
- `guides`: Guides and best practices
- `advanced`: Advanced usage scenarios (Circuit Breaker, Custom Errors)
- `examples`: Usage examples

### Spanish (`/es`)
- `es/api-reference`: Documentación de la API
- `es/concepts`: Conceptos fundamentales
- `es/guides`: Guías y buenas prácticas
- `es/advanced`: Escenarios avanzados
- `es/examples`: Ejemplos de uso
