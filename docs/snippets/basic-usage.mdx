## `run` para una tarea

```typescript
import { run } from "tryo";

const r = await run(() => fetch("/api").then((x) => x.json()));
if (r.ok) console.log(r.data);
else console.error(r.error.code);
```

## `all` para varias tareas

```typescript
import { all, isSuccess } from "tryo";

const tasks = [() => fetch("/a"), () => fetch("/b")];
const results = await all<Response>(tasks, { concurrency: 2 });

const successes = results.filter(isSuccess).map((r) => r.data);
```

## Reusar un runner

```typescript
import tryo from "tryo";
export const runner = tryo({ ignoreAbort: true });

const r = await runner.run(() => fetch("/api"));
```
