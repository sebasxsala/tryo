---
title: runAll
description: Executes multiple tasks and throws on the first failure
---

## Signature

`runAll<T>(tasks, options?): Promise<T[]>`

## Options

- Inherits all `run` options
- `concurrency`: simultaneous limit

## Example

```typescript
import { runAll } from "runtry";

const data = await runAll<string>(
  [
    () => fetch("/a").then((r) => r.text()),
    () => fetch("/b").then((r) => r.text()),
  ],
  { concurrency: 2 }
);
```

If a task fails, the normalized error is thrown.
