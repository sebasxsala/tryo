title: Concurrency
description: Control de concurrencia y modos de ejecución en lotes
---

## Opciones

- `concurrency`: límite de tareas simultáneas
- `mode`: en `runAllSettled`, `settle` o `fail-fast`

## `runAllSettled` con límite

```typescript
import { runAllSettled } from "runtry";

const tasks = Array.from({ length: 10 }, (_, i) => async () => i);

const results = await runAllSettled<number>(tasks, { concurrency: 3 });
```

## `fail-fast`

Detiene el inicio de nuevas tareas al primer error.

```typescript
const results = await runAllSettled<number>(tasks, { concurrency: 4, mode: "fail-fast" });
```

Las tareas no iniciadas quedan en estado `skipped`.

## `runAll`

Lanza al primer error y respeta `concurrency`.

```typescript
import { runAll } from "runtry";

const data = await runAll<number>(tasks, { concurrency: 2 });
```
